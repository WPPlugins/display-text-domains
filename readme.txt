Simple loading text domains loading mapper

== Description ==
This tool is proposed to debug loaded textdomains in plugins and themes. Also shows loaded files and all files wordpress is attemptiong to load for current translation. It may also help you loading translation issues caused by incorrect file permissions.

== Installation ==
Best is to install directly from WordPress. If manual installation is required, please make sure that the plugin files are in a folder named "text-domain-loadmap" (not two nested folders) in the WordPress plugins folder, usually "wp-content/plugins".

== Frequently Asked Questions ==
### How do I use this? ###
Go to Tools and look for the "Display text domains".
